# Wallpaper Digital Clock

A stylish and responsive **Digital Clock Web App** built using **HTML, CSS, and JavaScript**.  
This project displays **live time**, **current date**, **AM/PM format**, and features an **auto-changing wallpaper background** for a modern look.

---

## 🚀 Features

- ⏰ Live Digital Clock
- 📅 Current Date Display
- 🌗 12-Hour Format with AM/PM
- 🖼 Auto-Changing Background Wallpapers
- ✨ Smooth UI Design
- 📱 Responsive Layout
- 🌙 Dark Overlay for Better Visibility
